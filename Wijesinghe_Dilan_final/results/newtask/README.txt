Newtask

The controller used for the newtask run through is a Feedforward controller. 
The Feedback gains Kp and Ki are set to:     
Kp_test = 0.75
Ki_test = 0.005

The location of the cubes is set to:
Inital: (x,y) = 1, 0.5
Goal: (x,y) = 2, -2

Error Graphs are saved within the folder as png files.

X_err is saved in a csv called newtask_err.csv

The csv to run the code is contained in newtask.csv

The output of the code is stored in a file called newtask.log
